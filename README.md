# CatChat

A Chrome Extension for messenger.com (Facebook client just for Facebook messaging) that generates cat memes from a random assortment of cat pictures and recent facebook messages.

Created over Winter Break '15-'16 by
 - Max Yeo
 - Justin He
 - Andy Lao

Building
---
Go to chrome://extensions/ and enable "developer mode".  Then click "load unpacked extension" and select the folder containing all of this repository's files.  Then navigate to messenger.com
